Title: Multi-Dimensional Academic Intelligence System

 About Project
This project is about analyzing student performance using marks, attendance and assignment scores. 
Instead of using simple lists, I used structured data and did some calculations to understand the performance better.

  What I Did
- Generated student data using random values  
- Stored data using list and tuples  
- Converted data into DataFrame using pandas  
- Classified students into At Risk, Average, Good and Top Performer  
- Calculated mean, median, standard deviation and correlation  
- Normalized the marks  
- Gave final result based on conditions  

 Personalization
My roll number last digit is 7, so I generated 10 + 7 = 17 students.  
I also created my own formula for performance index:

(marks * 0.6 + assignment * 0.4) * log(attendance + 1)

This helps to combine marks, assignment and attendance into one value.

 How to Run
1. Install pandas and numpy  
2. Run the program  
3. Enter last digit of roll number  

 Output
The program shows:
- Student data in table format  
- Categories of students  
- Statistical values  
- Final system result  

 Conclusion
This project helped me understand how to use Python, pandas and numpy to analyze data and find useful insights from it.
